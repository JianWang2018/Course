/////////////////////////////////////////////////////////////////////////////
			Template Class Array 

/////////////////////////////////////////////////////////////////////////////

This program is built on Visual studio 2015 and includes two main files:
   1.  TemplateClassArray.h  
   2.  TemplateClassArray.cpp

TemplateClassArray.h
    this is a main header file, including all the declarations and definations of members in a class Array.

TemplateClassArray.cpp
    this is a main application source file and it is used to test all member functions and operators in Array for different types, such as int, float and double .

/////////////////////////////////////////////////////////////////////////////
Other standard files:

StdAfx.h, StdAfx.cpp
    These files are used to build a precompiled header (PCH) file
    named Array_op.pch and a precompiled types file named StdAfx.obj.
