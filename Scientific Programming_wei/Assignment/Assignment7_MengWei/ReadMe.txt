This program is built on Visual studio 2015 and includes two main files:
   1.  Array_op.h  
   2.  Array_op.cpp

Array_op.h
    this is a main header file, including all the declarations of members in a class Array.

Array_op.cpp
    this is a main application source file, including all the definitions of members and friend function in Array 
    and the main function which is used to test all member functions and operators in Array.

/////////////////////////////////////////////////////////////////////////////
Other standard files:

StdAfx.h, StdAfx.cpp
    These files are used to build a precompiled header (PCH) file
    named Array_op.pch and a precompiled types file named StdAfx.obj.
