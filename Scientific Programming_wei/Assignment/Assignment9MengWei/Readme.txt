/////////////////////////////////////////////////////////////////////////////
			SparseMatrix
/////////////////////////////////////////////////////////////////////////////

This program is built on Visual studio 2015 and includes two main files:
   1.  SparseMatrix.h  
   2.  SparseMatrix.cpp

SparseMatrix.h
    this is a main header file, including the construction of the struct term and all the declarations of members in a class SparseMatrix.

SparseMatrix.cpp
    this is a main application source file and it is used to define all memebers of the class SparseMatrix and to finish the storage and operations(+ & *) of 
 Sparse Matrix with whose element is characterized by <row, col, value>. 

/////////////////////////////////////////////////////////////////////////////
Other standard files:

StdAfx.h, StdAfx.cpp
    These files are used to build a precompiled header (PCH) file
    named Array_op.pch and a precompiled types file named StdAfx.obj.
